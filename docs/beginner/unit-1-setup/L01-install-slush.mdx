# L1: Install Slush Wallet

The first step to using the Sui blockchain is to install the Slush wallet.

## Overview

| Item | Content |
|:--|:--|
| **Goal** | Install the wallet |
| **Time** | ~5 minutes |
| **Tools** | Slush (Chrome Extension) |
| **Success Check** | Wallet shows your address |

## What is Slush Wallet?

Slush is a browser wallet for the Sui blockchain. It provides the following features:

- Generate and manage accounts (addresses)
- Sign transactions
- Connect to dApps
- Switch between networks (Mainnet/Testnet/Devnet)

:::tip Why Slush?
There are several wallets available for Sui, but this tutorial uses Slush. It's close to an official wallet and supports Sui-specific features like sponsored transactions.
:::

---

## Step 1: Install Chrome Extension

1. Go to the Chrome Web Store
   - [Slush - Chrome Web Store](https://chromewebstore.google.com/detail/slush-a-sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil)

2. Click the "Add to Chrome" button

3. Click "Add extension" in the confirmation dialog

![Install Chrome Extension](pathname:///img/beginner/unit-1/l01-chrome-extension.png)

:::note
Pin the extension to your toolbar for quick access.
:::

---

## Step 2: Create Account

1. After installation, click the Slush icon to launch it

2. Select "Create New Wallet"

3. Set a password
   - Use a strong password with at least 8 characters

4. A recovery phrase (12 words) will be displayed

![Create Account](pathname:///img/beginner/unit-1/l01-create-account.png)

:::danger Keep your recovery phrase safe
Never share your recovery phrase with anyone. Anyone with this phrase can access your wallet. We recommend writing it down on paper and storing it in a safe place.
:::

---

## Step 3: Confirm Backup

1. Write down the displayed recovery phrase

2. Click "Next" and enter the phrase in the correct order on the confirmation screen

3. Once confirmed, the wallet main screen will be displayed

---

## Success Check

You have succeeded if your **address** is displayed on the wallet main screen.

![Wallet Address](pathname:///img/beginner/unit-1/l01-wallet-address.png)

The address is a 64-character hexadecimal string starting with `0x` (e.g., `0x1234...abcd`).

:::tip Copy Address
Click on the address to copy it to your clipboard. You'll use this in later lessons.
:::

---

## Common Errors and Solutions

### "Extension won't install"

**Cause**: Chrome version is old, or conflict with other extensions

**Solution**:
1. Update Chrome to the latest version
2. Try in incognito mode
3. Temporarily disable other wallet extensions

### "Can't set password"

**Cause**: Password doesn't meet requirements

**Solution**:
- At least 8 characters
- Include letters and numbers
- Add special characters

### "Recovery phrase verification failed"

**Cause**: Wrong word order or spelling mistake

**Solution**:
1. Check the phrase again
2. Pay attention to spaces between words
3. Verify case (all lowercase)
