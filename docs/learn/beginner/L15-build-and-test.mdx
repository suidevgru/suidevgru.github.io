# Build and Test

:::info
This lesson is currently available in Japanese. English content coming soon.
:::

In this lesson, you'll add a `#[test]` function to `counter.move` and run `sui move test` to verify the increment logic locally.

## What you'll do

- Add a `#[test]` function to the counter module
- Run `sui move build` to confirm no compile errors
- Run `sui move test` to verify the increment logic passes

## Prerequisites

- Completed [L14: Write a Minimal Contract](/docs/learn/beginner/L14-write-minimal-contract)
- Sui CLI installed
