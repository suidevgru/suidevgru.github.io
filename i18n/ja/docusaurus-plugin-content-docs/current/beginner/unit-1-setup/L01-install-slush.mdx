# Slushウォレットを入れる

Suiを使うための最初のステップは、ウォレットのインストール。難しいことはありません｡画面の指示に従ってクリックしていくだけ！約5分で完了するよ。

## Slushウォレットって？

Slush（旧称「Sui Wallet」）は、Mysten Labs が提供するSui用のブラウザウォレット。これがあれば、いろんなことができるようになります｡<br/>
たとえば：

- アカウント（アドレス）の作成・管理
- トランザクションの署名
- dApp（分散型アプリ）との接続
- ネットワーク（Mainnet/Testnet など）の切り替え

:::tip なぜSlush？
Suiには複数のウォレットがあるけど、このチュートリアルではSlushを使うよ！公式に近いウォレットで、Sui特有の機能（スポンサードトランザクションなど）にも対応してるんだ。
:::

---

## Chrome拡張をインストールしよう

まずはChromeにSlushを追加するよ。

1. Chrome Web Storeにアクセス
   - [Slush - Chrome Web Store](https://chromewebstore.google.com/detail/slush-a-sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil)

2. 開発元が「✓ slush.app」になっていることを確認し､「**Chromeに追加**」をクリック

   ![Chrome Web Store](./img/L1_slush-chrome-store_ja.png)

3. 確認ダイアログで「拡張機能を追加」をクリック

:::note
ツールバーにピン留めしておくと、すぐにアクセスできて便利だよ！
:::

---

## アカウントを作成しよう

Slushには「ソーシャルログイン」（Google/Apple/Facebook/Twitch）と「パスフレーズ」の2つの方式があるよ。ここでは、後から他のウォレットにも移行しやすい**パスフレーズ方式**で進めるね。

1. インストールが完了すると、Slushのウェルカム画面が自動で開くよ

2. 画面下部の「More options」をクリック

   ![Slushウェルカム画面](./img/L1_create-wallet-1.png)

3. 「Create a passphrase account」を選択

   ![パスフレーズアカウント作成](./img/L1_create-wallet-2.png)

---

## パスフレーズを保管しよう

1. 12個の英単語（パスフレーズ）が表示されるよ

2. これを紙に書き留めておこう

:::danger パスフレーズは大切に保管してね
パスフレーズは絶対に他人に教えちゃダメ！これがあれば誰でもあなたのウォレットにアクセスできちゃうんだ。紙に書いて、安全な場所に保管しておこう。
:::

---

## パスワードとセキュリティを設定しよう

1. パスワードを設定するよ
   - 英数字＋記号を含む、できるだけ長いパスワードがおすすめ

2. `Protect your wallet with additional security`画面が表示されるよ｡<br/>
   お好みで設定して「Next」をクリック
   - **Unlock your wallet**: 自動ロックまでの時間（15分〜1日）
   - **Confirm transactions**: トランザクションごとにパスワードを要求するか

3. これでウォレットのメイン画面が表示されるよ

---

## できたかな？

ウォレットの`Settings`を開いて、**アドレス**が表示されていれば成功！

![ウォレットアドレス表示](./img/L1_confarm-address.png)

アドレスは `0x` で始まる長い文字列だよ（例: `0x1234...abcd`）。

:::tip アドレスをコピーしておこう
アドレス右側のコピーボタンをクリックするとクリップボードにコピーされるよ。後のレッスンで使うから、メモしておくと便利！
:::
