# Slushウォレットを入れる

Suiブロックチェーンを使うための最初のステップとして、Slushウォレットをインストールします。

## 概要

| 項目 | 内容 |
|:--|:--|
| **Goal** | ウォレットをインストールする |
| **所要時間** | 約5分 |
| **使用ツール** | Slush（Chrome拡張） |
| **Success Check** | ウォレットにアドレスが表示される |

## Slushウォレットとは

SlushはSuiブロックチェーン用のブラウザウォレットです。以下の機能を提供します：

- アカウント（アドレス）の生成と管理
- トランザクションの署名
- dAppとの接続
- 複数ネットワーク（Mainnet/Testnet/Devnet）の切り替え

:::tip なぜSlush？
Suiには複数のウォレットがありますが、このチュートリアルではSlushを使用します。公式に近いウォレットで、Sui特有の機能（スポンサードトランザクションなど）に対応しています。
:::

---

## Step 1: Chrome拡張をインストール

1. Chrome Web Storeにアクセスします
   - [Slush - Chrome Web Store](https://chromewebstore.google.com/detail/slush-a-sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil)

2. 「Chromeに追加」ボタンをクリックします

3. 確認ダイアログで「拡張機能を追加」をクリックします

![Chrome拡張のインストール](pathname:///img/beginner/unit-1/l01-chrome-extension.png)

:::note
拡張機能をピン留めすると、ツールバーからすぐにアクセスできます。
:::

---

## Step 2: アカウントを作成

1. インストール後、Slushアイコンをクリックして起動します

2. 「Create New Wallet」を選択します

3. パスワードを設定します
   - 8文字以上の強力なパスワードを使用してください

4. リカバリーフレーズ（12単語）が表示されます

![アカウント作成画面](pathname:///img/beginner/unit-1/l01-create-account.png)

:::danger リカバリーフレーズを安全に保管
リカバリーフレーズは絶対に他人に教えないでください。これがあれば誰でもあなたのウォレットにアクセスできます。紙に書いて安全な場所に保管することを推奨します。
:::

---

## Step 3: バックアップを確認

1. 表示されたリカバリーフレーズを書き留めます

2. 「次へ」をクリックし、確認画面でフレーズを正しい順序で入力します

3. 確認が完了すると、ウォレットのメイン画面が表示されます

---

## Success Check

ウォレットのメイン画面に**アドレス**が表示されていれば成功です。

![ウォレットアドレス表示](pathname:///img/beginner/unit-1/l01-wallet-address.png)

アドレスは `0x` で始まる64文字の16進数文字列です（例: `0x1234...abcd`）。

:::tip アドレスのコピー
アドレスをクリックするとクリップボードにコピーされます。後のレッスンで使用します。
:::

## Common Errors and Solutions

### 「拡張機能が追加できない」

**原因**: Chromeのバージョンが古い、または他の拡張機能との競合

**解決策**:
1. Chromeを最新版にアップデート
2. シークレットモードで試す
3. 他のウォレット拡張機能を一時的に無効化

### 「パスワードが設定できない」

**原因**: パスワードが要件を満たしていない

**解決策**:
- 8文字以上
- 英数字を含める
- 特殊文字を追加

### 「リカバリーフレーズの確認に失敗」

**原因**: 単語の順序が間違っている、またはスペルミス

**解決策**:
1. フレーズを再度確認
2. 単語間のスペースに注意
3. 大文字小文字を確認（すべて小文字）
