# はじめよう

ようこそ！このセクションでは、Suiで開発を始めるために必要なものをすべてセットアップします。心配いりません—各ステップはシンプルで、一つずつ丁寧にガイドします。

## このセクションで学ぶこと

このセクションを完了すると、以下が準備できています：

- Slushウォレットがインストールされ、アドレスが表示される
- ウォレットがDevnetに接続されている
- Sui CLIがインストールされ、Devnetに接続されている
- Faucetからテストトークンを取得できる

## このセクションのレッスン

1. [**Slushウォレットを入れる**](./L01-install-slush.mdx) – 最初のSuiウォレットをセットアップ
2. **Devnetに切り替える** – 開発ネットワークに接続
3. **Sui CLIをインストールする** – コマンドラインツールを導入
4. **VSCode + Sui Extension** – Move開発環境を整える
5. **CLIをDevnetに接続する** – CLIのネットワーク設定を行う
6. **Faucetでトークンを取得する** – テストSUIを入手

## 完了の確認方法

以下を確認できれば、このセクションは完了です：

- `sui client active-env` が `devnet` を返す
- `sui client gas` で残高が表示される

## 使用するツール

- **Slush**: Suiウォレット（ブラウザ拡張）
- **Sui CLI**: コマンドラインツール
- **Sui Extension**: VSCodeのMove開発拡張
