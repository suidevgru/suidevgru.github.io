# Slushウォレットをインストール

Suiを使うための最初のステップは、ウォレットのインストールです。難しいことはありません。画面の指示に従ってクリックしていくだけで、約5分で完了します。

## Slushウォレットとは

Slush（旧称「Sui Wallet」）は、Mysten Labs が提供するSui用のブラウザウォレットです。Slushを使うと、以下のようなことができます。

- アカウント（アドレス）の作成・管理
- トランザクションの署名
- dApp（分散型アプリ）との接続
- ネットワーク（Mainnet/Testnet など）の切り替え

:::tip なぜSlush？
Suiには複数のウォレットがありますが、このチュートリアルではSlushを使います。公式に近いウォレットで、Sui特有の機能（スポンサードトランザクションなど）にも対応しています。
:::

---

## Chrome拡張機能をインストールする

まずはChromeにSlushを追加します。

1. Chrome Web Storeにアクセス
   - [Slush - Chrome Web Store](https://chromewebstore.google.com/detail/slush-a-sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil)

2. 開発元が「✓ slush.app」になっていることを確認し､「**Chromeに追加**」をクリック

   ![Chrome Web Store](./img/L1_slush-chrome-store_ja.png)

3. 確認ダイアログで「拡張機能を追加」をクリック

:::note
ツールバーにピン留めしておくと、すぐにアクセスできて便利です。
:::

---

## アカウントを作成する

Slushには「ソーシャルログイン」（Google/Apple/Facebook/Twitch）とリカバリーフレーズ方式の2つがあります。ここでは、後から他のウォレットにも移行しやすい**リカバリーフレーズ方式**(Passphrase account)で進めます。

1. インストールが完了すると、Slushのウェルカム画面が自動で開きます

2. 画面下部の「More options」をクリック

   ![Slushウェルカム画面](./img/L1_create-wallet-1.png)

3. 「Create a passphrase account」を選択

   ![リカバリーフレーズアカウント作成](./img/L1_create-wallet-2.png)

---

## リカバリーフレーズを保管する

1. 12個の英単語（リカバリーフレーズ、シードフレーズ）が表示されます

2. 紙に書き留めて安全に保管します

:::danger リカバリーフレーズを安全に保管してください
リカバリーフレーズ（シードフレーズ）はウォレットの「鍵」です。

- 絶対に他人に教えないでください。
- スクリーンショットを撮ったり、チャット・メール・クラウドメモに保存しないでください。
- 紙に書いてオフラインで安全に保管してください。
:::

---

## パスワードとセキュリティを設定する

1. パスワードを設定します
   - 英数字＋記号を含む、できるだけ長いパスワードを推奨します

2. `Protect your wallet with additional security`画面が表示されます。<br/>
   お好みで設定して「Next」をクリックします
   - **Unlock your wallet**: 自動ロックまでの時間（15分〜1日）
   - **Confirm transactions**: トランザクションごとにパスワードを要求するか

3. 設定が完了すると、ウォレットのメイン画面が表示されます

---

## 成功の確認

ウォレットの`Settings`（設定）を開いて、**アドレス**が表示されていれば成功です。

![ウォレットアドレス表示](./img/L1_confarm-address.png)

アドレスは `0x` で始まる長い文字列です（例: `0x1234...abcd`）。

:::tip アドレスをコピーしておく
アドレス右側のコピーボタンをクリックするとクリップボードにコピーされます。次のレッスンで使うので、必要なときにすぐ見つけられるようにしておくと便利です。
:::
