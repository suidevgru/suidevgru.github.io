# ウォレットをPTB Builderに接続する

このレッスンでは、SlushウォレットをPTB Builderに接続します。**難しくありません**。
手順どおりに進めれば、数分で完了します。

## PTB Builderとは

**PTB Builder**は、Suiのトランザクション（Programmable Transaction Block / PTB）を視覚的に構築・実行できる開発者向けツールです。

通常、トランザクションを作成するにはCLIやSDKを使って操作する必要がありますが、PTB Builderを使えば、ブロックをドラッグ＆ドロップでつなげるだけでトランザクションを組み立てられます。

ウォレットを接続することで、

- 自分のウォレットアドレスを使って
- 作成したトランザクションに署名し
- 実際にブロックチェーン上で実行する

といった操作が可能になります。

:::tip PTB builderでのトランザクション作成の流れ
1. **ウォレットを接続**（このレッスン）
2. トランザクションを視覚的に作成
3. 署名して実行
:::

---

## 前提条件

このレッスンを始める前に、以下が完了していることを確認してください。

- [Slushウォレットをインストール](/docs/getting-started/L01-install-slush) 済み
- [Devnetに切り替え](/docs/getting-started/L02-switch-devnet) 済み

---

## 手順

### 1. PTB Builderを開く

以下のリンクからPTB Builderにアクセスします。

→ [PTB Builder（ptb.wal.app）](https://ptb.wal.app/)

ページを開くと、画面中央にウォレット接続画面が表示されます。

### 2. ネットワークを選択

画面中央のドロップダウンから使用したいネットワークを選択します｡
このチュートリアルでは「**DEVNET**」を選択します。

:::warning ネットワークの一致
選択するネットワークは、Slushウォレットのネットワークと一致させましょう。
このチュートリアルでは両方とも **Devnet** を使用します。
:::

### 3. 「Connect Wallet」をクリック

画面中央の「**Connect Wallet**」ボタンをクリックします。

### 4. Slushを選択

ウォレット選択画面が表示されます。
一覧から「**Slush**」を選択します。

:::note 他のウォレット
Sui Wallet、Suiet、Ethos Walletなど、他のSui対応ウォレットも選択できます。
このチュートリアルでは **Slushウォレット** を使用します。
:::

---

### 5. 接続を承認

Slushウォレットのポップアップが表示されます。
パスワードを求められた場合は入力し、接続したいウォレットアドレスを選択して「**Approve**」をクリックします。

接続が完了すると、「Connect Wallet」ボタンが接続したウォレットアドレスに変わります。

```text
例: 0xe8d3...04ef
```

---

### 6. Editor / Viewer を選択

画面下部に2つのボタンがあります。

- **Editor**: 新しいトランザクションを視覚的に構築する
- **Viewer**: 実行済みトランザクションの流れを視覚的に確認する

このチュートリアルでは「**Editor**」をクリックして、PTB Builderのメインエディタ画面に進みます。

---

## 成功の確認

エディタ画面が表示されれば接続成功です。

:::info ネットワークの確認
PTB Builder（ptb.wal.app）は **Mainnet**、**Testnet**、**Devnet** に対応しています。
このチュートリアルでは **Devnet** を使用しています。

PTB Builderのネットワーク選択とSlushウォレットのネットワークが一致していないと、
トランザクションの作成や実行に失敗することがあります。
:::
