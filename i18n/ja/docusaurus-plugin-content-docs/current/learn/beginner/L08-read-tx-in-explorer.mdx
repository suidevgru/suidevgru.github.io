# Explorerでトランザクションを読む

このレッスンでは、ブロックチェーンエクスプローラーを使ってトランザクションの詳細を確認する方法を学びます。**難しくありません**。ブロックチェーン上で何が起きたのかを自分の目で確認できるようになりましょう。

:::tip なぜエクスプローラーを使うの？
開発中に「トランザクションが成功したはずなのに期待した動作をしない」ということがよくあります。
エクスプローラーで実際に何が起きたかを確認することで、問題の原因を特定できます。
:::

## エクスプローラーとは？

**ブロックチェーンエクスプローラー** は、ブロックチェーン上のすべてのトランザクション、アドレス、オブジェクトを検索・閲覧できるツールです。

Suiには複数のエクスプローラーがあります：

| エクスプローラー | URL |
|-----------------|-----|
| **Suiscan** | [suiscan.xyz](https://suiscan.xyz/) |
| **SuiVision** | [suivision.xyz](https://suivision.xyz/) |

どちらを使っても基本的な機能は同じですが、**本サイトでは Suiscan を使用します**。

Suiscanを使うと：

- 自分が送ったトランザクションの詳細を確認できる
- 他のアドレスの取引履歴を見ることができる
- スマートコントラクト（Package）の内容を調べられる

## 手順

### 1. Suiscanを開く

以下のリンクからSuiscanにアクセスします。

→ [Suiscan (suiscan.xyz)](https://suiscan.xyz/)

{/* ![Suiscanホーム画面](./img/L08_home.png) */}

### 2. ネットワークをDevnetに切り替える

画面右上のネットワーク選択から「**Devnet**」を選択します。

{/* ![ネットワーク選択](./img/L08_network-select.png) */}

:::warning ネットワークを必ず確認
Mainnet/Testnet/Devnetで別々のブロックチェーンです。
ウォレットと同じネットワーク（Devnet）を選択してください。
:::

### 3. トランザクションを検索する

トランザクションを検索するには、以下のいずれかを使います：

- **Transaction Digest**: トランザクション固有のID（例: `ABC123...xyz`）
- **アドレス**: 自分のウォレットアドレス

#### 方法A: Transaction Digestで検索

PTB BuilderやCLIでトランザクションを実行すると、Transaction Digestが表示されます。
これを検索バーに貼り付けて検索します。

#### 方法B: アドレスで検索

自分のウォレットアドレスを検索バーに入力すると、そのアドレスに関連する全てのトランザクション履歴が表示されます。

{/* ![アドレス検索結果](./img/L08_search-address.png) */}

### 4. トランザクションの詳細を読む

トランザクションを開くと、以下の情報が表示されます：

{/* ![トランザクション詳細画面](./img/L08_tx-detail.png) */}

#### Overview（概要）

| 項目 | 説明 |
|------|------|
| **Status** | Success（成功）/ Failure（失敗） |
| **Timestamp** | 実行日時 |
| **Sender** | トランザクションを送信したアドレス |
| **Transaction Digest** | トランザクションの固有ID |

#### Input / Output（入出力）

- **Input**: トランザクションで使用されたオブジェクト（コイン、NFTなど）
- **Output**: トランザクションの結果として作成/変更されたオブジェクト

{/* ![Input/Outputセクション](./img/L08_input-output.png) */}

:::tip Input/Outputの読み方
- Inputに自分のコインが表示され、Outputに減った残高が表示されていれば、送金が成功しています
- 新しいオブジェクトがOutputに表示されていれば、何かが作成されたことを意味します
:::

#### Gas（ガス代）

| 項目 | 説明 |
|------|------|
| **Gas Budget** | 設定した最大ガス上限 |
| **Gas Used** | 実際に使用されたガス量 |
| **Gas Price** | ガス単価 |

上記のトランザクション詳細画像では、「Total Gas Fee」と「Gas Budget」が Overview セクションに表示されています。

### 5. PTBの詳細を確認する（オプション）

PTB（Programmable Transaction Block）を実行した場合、「**Transaction Block**」タブで各操作の詳細を確認できます。

- どのような操作（Transfer, MoveCall等）が実行されたか
- 各操作の引数は何だったか
- 操作の順序

## よくあるトラブル

### トランザクションが見つからない

- ネットワーク（Devnet/Testnet/Mainnet）が正しいか確認してください
- Transaction Digestをコピーする際に余分な空白が入っていないか確認してください
- トランザクション実行から反映まで数秒かかることがあります

### Failedと表示される

- ガス不足の場合、Faucetで追加のSUIを取得してください
- エラーメッセージを確認し、何が原因かを特定してください
