# ウォレットをPTB Builderに接続する

このレッスンでは、SlushウォレットをPTB Builderに接続します。**難しくありません**。手順どおりに進めれば数分で完了します。

## PTB Builderとは

**PTB Builder**は、Suiのトランザクション（Programmable Transaction Block）を視覚的に構築できるツールです。

通常、トランザクションを作成するにはコードを書く必要がありますが、PTB Builderを使えば、ブロックをドラッグ＆ドロップでつなげるだけでトランザクションを組み立てられます。

ウォレットを接続することで、PTB Builderで作成したトランザクションに署名し、実際にブロックチェーン上で実行できるようになります。

:::tip このセクションでの流れ
1. **ウォレットを接続**（このレッスン）
2. トランザクションを視覚的に作成
3. 署名して実行
:::

---

## 前提条件

このレッスンを始める前に、以下が完了していることを確認してください。

- [Slushウォレットをインストール](/docs/getting-started/L01-install-slush)済み
- [Devnetに切り替え](/docs/getting-started/L02-switch-devnet)済み

---

## 手順

### 1. PTB Builderページを開く

サイドバーから「**Live Code Example**」>「**PTB Builder**」を選択するか、以下のリンクからアクセスします。

→ [PTB Builder デモページ](/live-code-example/ptb-builder)

ページを開くと、PTB Builderのインターフェースが表示されます。

<!-- 画像: PTB Builderの初期画面 -->
<!-- ![PTB Builder初期画面](./img/L07_ptb-builder-initial.png) -->

画面上部に以下の情報が表示されています：

- **Network**: `sui:devnet`（接続先のネットワーク）
- **Address**: `— (connect wallet)`（ウォレット未接続の状態）

---

### 2. 「Connect Wallet」をクリック

画面右上の「**Connect Wallet**」ボタンをクリックします。

<!-- 画像: Connect Walletボタンの位置 -->
<!-- ![Connect Walletボタン](./img/L07_connect-wallet-button.png) -->

---

### 3. Slushを選択

ウォレット選択画面が表示されます。一覧から「**Slush**」を選択します。

<!-- 画像: ウォレット選択画面 -->
<!-- ![ウォレット選択](./img/L07_select-slush.png) -->

:::note 他のウォレット
Sui Wallet、Suiet、Ethos Walletなど、他のSui対応ウォレットも選択できます。このチュートリアルではSlushを使用します。
:::

---

### 4. 接続を承認

Slushのポップアップが表示されます。「**Connect**」をクリックして接続を承認します。

<!-- 画像: Slush接続承認画面 -->
<!-- ![接続承認](./img/L07_approve-connection.png) -->

:::tip パスワードの入力
Slushの設定によっては、パスワードの入力を求められる場合があります。
:::

---

## 成功の確認

接続が成功すると、画面上部の**Address**欄にあなたのウォレットアドレスが表示されます。

<!-- 画像: 接続成功後の画面 -->
<!-- ![接続成功](./img/L07_connected.png) -->

```
Network: sui:devnet
Address: 0x1234...abcd（あなたのアドレス）
```

「**Connect Wallet**」ボタンが、接続中のアドレス表示に変わっていれば成功です。

:::info ネットワークの確認
PTB Builderは**Devnet**のみ対応しています。Slushウォレットで別のネットワーク（MainnetやTestnet）を選択している場合でも、PTB BuilderからはDevnet上でトランザクションを実行します。
:::
