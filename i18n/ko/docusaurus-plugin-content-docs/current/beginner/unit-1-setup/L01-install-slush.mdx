# Slush 지갑 설치하기

Sui를 사용하기 위한 첫 번째 단계는 지갑 설치야. 어렵지 않아! 화면 안내에 따라 클릭만 하면 돼. 약 5분이면 완료!

## Slush 지갑이 뭐야?

Slush(이전 명칭 "Sui Wallet")는 Mysten Labs에서 만든 Sui용 브라우저 지갑이야. 이게 있으면 여러 가지를 할 수 있어.<br/>
예를 들면:

- 계정(주소) 생성 및 관리
- 트랜잭션 서명
- dApp(분산형 앱)과 연결
- 네트워크(Mainnet/Testnet 등) 전환

:::tip 왜 Slush야?
Sui에는 여러 지갑이 있지만, 이 튜토리얼에서는 Slush를 사용해! 공식에 가까운 지갑이고, Sui 고유의 기능(스폰서 트랜잭션 등)도 지원하거든.
:::

---

## Chrome 확장을 설치하자

먼저 Chrome에 Slush를 추가할 거야.

1. Chrome Web Store에 접속
   - [Slush - Chrome Web Store](https://chromewebstore.google.com/detail/slush-a-sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil)

2. 개발자가 "✓ slush.app"인지 확인하고 "**Chrome에 추가**" 클릭

   ![Chrome Web Store](./img/L1_slush-chrome-store_en.png)

3. 확인 대화상자에서 "확장 프로그램 추가" 클릭

:::note
도구 모음에 고정해 두면 바로 접근할 수 있어서 편해!
:::

---

## 계정을 만들자

Slush에는 "소셜 로그인"(Google/Apple/Facebook/Twitch)과 "패스프레이즈" 두 가지 방식이 있어. 여기서는 나중에 다른 지갑으로 이전하기 쉬운 **패스프레이즈 방식**으로 진행할게.

1. 설치가 완료되면 Slush 환영 화면이 자동으로 열려

2. 화면 하단의 "More options" 클릭

   ![Slush 환영 화면](./img/L1_create-wallet-1.png)

3. "Create a passphrase account" 선택

   ![패스프레이즈 계정 만들기](./img/L1_create-wallet-2.png)

---

## 패스프레이즈를 저장하자

1. 12개의 영어 단어(패스프레이즈)가 표시돼

2. 이걸 종이에 적어두자

:::danger 패스프레이즈를 안전하게 보관해!
패스프레이즈는 절대로 다른 사람에게 알려주면 안 돼! 이게 있으면 누구나 네 지갑에 접근할 수 있거든. 종이에 적어서 안전한 곳에 보관해.
:::

---

## 비밀번호와 보안을 설정하자

1. 비밀번호를 설정해
   - 영숫자 + 기호를 포함해서 최대한 길게 만드는 게 좋아

2. `Protect your wallet with additional security` 화면이 표시돼.<br/>
   원하는 대로 설정하고 "Next" 클릭
   - **Unlock your wallet**: 자동 잠금까지의 시간(15분~1일)
   - **Confirm transactions**: 트랜잭션마다 비밀번호 요구 여부

3. 이제 지갑 메인 화면이 나타나!

---

## 잘 됐어?

지갑의 `Settings`를 열어서 **주소**가 표시되면 성공!

![지갑 주소 표시](./img/L1_confarm-address.png)

주소는 `0x`로 시작하는 긴 문자열이야 (예: `0x1234...abcd`).

:::tip 주소를 복사해 두자
주소 옆의 복사 버튼을 클릭하면 클립보드에 복사돼. 다음 레슨에서 사용하니까 메모해 두면 편해!
:::
