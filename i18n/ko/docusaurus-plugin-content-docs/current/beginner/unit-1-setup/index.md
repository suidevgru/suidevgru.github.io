# Unit 1: 환경 설정

Sui 개발 환경을 구축하고, Devnet에 연결하여 Faucet에서 테스트 토큰을 받는 방법을 배웁니다.

## Goal

이 Unit을 완료하면 다음을 할 수 있게 됩니다:

- Slush 지갑이 설치되어 있고 주소가 표시됩니다
- 지갑이 Devnet에 연결되어 있습니다
- Sui CLI가 설치되어 있고 Devnet에 연결되어 있습니다
- Faucet에서 테스트 토큰을 받을 수 있습니다

## Lessons

| Lesson | Title | Goal |
|:--|:--|:--|
| [L1](./L01-install-slush.mdx) | Slush 지갑 설치하기 | 지갑을 설치합니다 |
| L2 | Devnet으로 전환하기 | 네트워크를 Devnet으로 변경합니다 |
| L3 | Sui CLI 설치하기 | 커맨드라인 도구를 설치합니다 |
| L4 | VSCode + Sui Extension | Move 개발 환경을 설정합니다 |
| L5 | CLI를 Devnet에 연결하기 | CLI 네트워크 설정을 합니다 |
| L6 | Faucet에서 토큰 받기 | 테스트 SUI를 받습니다 |

## Checkpoint

다음을 확인할 수 있으면 Unit 1이 완료됩니다:

- `sui client active-env`가 `devnet`을 반환합니다
- `sui client gas`로 잔액이 표시됩니다

## 사용 도구

- **Slush**: Sui 지갑 (브라우저 확장)
- **Sui CLI**: 커맨드라인 도구
- **Sui Extension**: VSCode의 Move 개발 확장
