# Devnet으로 전환하기

지갑 준비가 완료되었다면, 이제 개발용 네트워크로 전환해 봅시다. 2분도 걸리지 않는 간단한 작업입니다.

## Devnet이란?

Sui에는 여러 네트워크가 있습니다.

- **Mainnet** — 실제 환경입니다. 실제 자산을 다룹니다.
- **Testnet** — 검증 환경입니다. 실제 배포 전 동작 확인과 통합 테스트에 적합합니다.
- **Devnet** — 개발 환경입니다. 자유롭게 시행착오를 거치며 개발을 진행하기에 적합합니다.

이 튜토리얼에서는 **Devnet**을 사용하여 학습을 진행합니다.

Devnet은 개발과 학습을 목적으로 한 네트워크로, 테스트용 SUI(가스)를 **무료로 몇 번이든 받을 수** 있습니다.
실제로 조작하면서 시행착오를 거칠 수 있어, 처음 Sui를 접하는 분도 안심하고 진행할 수 있습니다.

참고로 **Mainnet**은 실제 가치를 가진 진짜 SUI를 다루는 실제 환경입니다.
이 튜토리얼에서는 사용하지 않으니 안심하고 Devnet으로 진행하세요.

:::tip 왜 Devnet을 사용하나요?
Devnet은 Sui CLI의 Faucet에서 테스트용 SUI(gas)를 쉽게 받을 수 있어 학습에 적합합니다.<br/>
(반면, Testnet의 Faucet은 접속 집중이나 속도 제한으로 인해 바로 받지 못하고 대기 시간이 발생할 수 있습니다.)<br/>
※ Devnet은 비교적 자주 데이터가 초기화됩니다. 장기 검증이나 결과물을 보존하려면 Testnet을 사용하세요.
:::

## Faucet이란?

**Faucet**(포셋)은 Devnet이나 Testnet에서 가스비로 사용할 테스트 토큰을 무료로 받을 수 있는 서비스입니다.
[Sui Faucet](https://faucet.sui.io/)이나 CLI에서 받을 수 있습니다.

Faucet 사용법은 [테스트 토큰 받기](./L06-get-test-tokens)에서 자세히 설명합니다.

## Devnet으로 전환하기

Slush 지갑에서 네트워크를 전환합니다.

### 1. Settings 열기

1. 브라우저 도구 모음에서 Slush 아이콘을 클릭하여 지갑을 엽니다<br/>
(비밀번호를 요청하면 입력하세요)

2. 오른쪽 하단의 사용자 아이콘을 클릭하여 "**Settings**" 화면을 엽니다

![Settings 메뉴](./img/L2_wallet_userbotton.png)

### 2. Devnet 선택하기

1. "**Network**"를 클릭합니다

2. "**Devnet**"을 선택합니다

![Devnet 선택](./img/L2_select-devnet.png)

## 성공 확인

지갑의 메인 화면으로 돌아가서 상단에 "**Devnet**"이 표시되면 성공입니다!
