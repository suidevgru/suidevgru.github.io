# 테스트 토큰 받기

[CLI를 Devnet에 연결하기](./L05-connect-cli-to-devnet.mdx)에서 Devnet 연결과 주소 생성을 완료했습니다. 이제 해당 주소에 테스트 토큰을 받아봅시다.

Sui에서 트랜잭션을 실행하려면 가스비로 SUI 토큰이 필요합니다. Devnet과 Testnet에서는 **Faucet**을 사용해 무료로 테스트 토큰을 받을 수 있습니다.

## Faucet이란?

[Devnet으로 전환하기](./L02-switch-devnet.mdx)에서 간단히 언급했듯이, **Faucet**은 테스트용 SUI 토큰을 무료로 배포하는 서비스입니다.

Devnet Faucet은 제한이 비교적 느슨해서 필요할 때마다 추가로 테스트 토큰을 받을 수 있습니다. 학습이나 개발 중 잔액이 부족하면 언제든 추가로 받을 수 있으니 걱정하지 마세요!

:::tip Devnet Faucet은 편하게 사용 가능
Testnet Faucet은 트래픽 집중이나 속도 제한으로 대기 시간이 발생할 수 있지만, Devnet Faucet은 비교적 빠르게 받을 수 있습니다.
:::

## CLI로 테스트 토큰 받기

**가장 간단한 방법**입니다. 먼저 Devnet에 연결되어 있는지 확인합시다.

```bash
sui client active-env
```

`devnet`이 표시되면 OK입니다. 다른 네트워크가 표시된 경우 `sui client switch --env devnet`으로 전환하세요.

확인되면 다음 명령어를 실행해 테스트 토큰을 받습니다.

```bash
sui client faucet
```

성공하면 다음과 같은 메시지가 표시됩니다.

```
Request successful. It can take up to 1 minute to get the coin. Run sui client gas to check your gas coins.
```

:::note 토큰 도착까지 잠시 대기
Faucet에서 토큰 전송은 즉시 이루어지지 않습니다. 보통 몇 초에서 1분 정도 걸립니다.
:::

## 잔액 확인하기

테스트 토큰이 도착했는지 확인해봅시다.

```bash
sui client gas
```

다음과 같이 표시되면 성공입니다.

```
╭────────────────────────────────────────────────────────────────────┬────────────────────┬──────────────────╮
│ gasCoinId                                                          │ mistBalance (MIST) │ suiBalance (SUI) │
├────────────────────────────────────────────────────────────────────┼────────────────────┼──────────────────┤
│ 0x*****************************************************************│ 10000000000        │ 10.00            │
╰────────────────────────────────────────────────────────────────────┴────────────────────┴──────────────────╯
```

:::info 코인 오브젝트란?
Sui에서는 모든 토큰이 "코인 오브젝트"라는 단위로 관리됩니다.

따라서 잔액은 "총 금액"이 아니라,
"10 SUI 코인이 2개 있다"와 같이 여러 코인 오브젝트로 존재할 수 있습니다.

SUI 코인은 트랜잭션의 가스비로 사용됩니다.

SUI의 최소 단위는 **MIST**이며,
**1 SUI = 1,000,000,000 MIST**입니다.
:::

:::tip Slush 지갑 잔액에 대해
CLI로 받은 테스트 토큰은 CLI에서 만든 주소에 연결되어 있습니다.
Slush 지갑을 설치할 때 만든 주소와는 다르기 때문에 Slush의 잔액은 증가하지 않습니다.

CLI 주소를 Slush에서도 사용하려면 [CLI를 Devnet에 연결하기](./L05-connect-cli-to-devnet.mdx)의 "Slush에 가져오는 방법 (선택 사항)"을 참조하세요.
:::

## Web Faucet 사용 방법

CLI 대신 웹 브라우저에서 받을 수도 있습니다.

1. [Sui Faucet](https://faucet.sui.io/)에 접속합니다

2. 네트워크를 **Devnet**으로 설정합니다

3. CLI 주소를 입력합니다
   - 주소는 `sui client active-address`로 확인할 수 있습니다

4. "**Request Tokens**"를 클릭합니다

몇 초에서 1분 정도면 토큰이 도착합니다.

:::note Testnet에서는 Web Faucet 사용
현재 Testnet에서는 `sui client faucet`을 실행하면 Web Faucet 사용이 안내됩니다.
테스트 토큰은 Web Faucet에서 받으세요.
:::
