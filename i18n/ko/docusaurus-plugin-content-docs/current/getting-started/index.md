# 시작하기

환영합니다! 이 섹션에서는 Sui에서 개발을 시작하기 위해 필요한 모든 것을 설정합니다. 걱정하지 마세요—각 단계는 간단하고, 하나씩 친절하게 안내해 드리겠습니다.

## 이 섹션에서 배울 내용

이 섹션을 완료하면 다음을 갖추게 됩니다:

- Slush 지갑이 설치되어 있고 주소가 표시됩니다
- 지갑이 Devnet에 연결되어 있습니다
- Sui CLI가 설치되어 있고 Devnet에 연결되어 있습니다
- Faucet에서 테스트 토큰을 받을 수 있습니다

## 이 섹션의 레슨

1. [**Slush 지갑 설치하기**](./L01-install-slush.mdx) – 첫 번째 Sui 지갑을 설정
2. [**Devnet으로 전환하기**](./L02-switch-devnet.mdx) – 개발 네트워크에 연결
3. [**Sui CLI 설치하기**](./L03-install-sui-cli.mdx) – 커맨드라인 도구를 설치
4. **VSCode + Sui Extension** – Move 개발 환경을 설정
5. **CLI를 Devnet에 연결하기** – CLI 네트워크 설정을 수행
6. **Faucet에서 토큰 받기** – 테스트 SUI를 받기

## 완료 확인 방법

다음을 확인할 수 있으면 이 섹션이 완료됩니다:

- `sui client active-env`가 `devnet`을 반환합니다
- `sui client gas`로 잔액이 표시됩니다

## 사용하는 도구

- **Slush**: Sui 지갑 (브라우저 확장 프로그램)
- **Sui CLI**: 커맨드라인 도구
- **Sui Extension**: VSCode의 Move 개발 확장 프로그램
